Project Description

This project was done in a group of 5, as an assignment to a module I took in AY2021/2022 Semester 2, which was BT2102 (Data Management and Visualisation).

The objective of this project was to design a graphical user interface to create an ecommerce system supporting various user functions such as user registration, user authentication, item purchases and item warranty services. The features are illustrated in the video above.

We used Python Tkinter for our graphical user interface, MySQL for our back-end, entity relationship modelling concepts for database design, and Git for version control.
