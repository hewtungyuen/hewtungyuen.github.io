My Internship Experience

Task 1: web scraping of URA circulars

The first task that I worked on was to scrape URA circular information from this website, storing the text information as an archive for future usage. The steps that I followed were as follows:

  1. Scrape data using BeautifulSoup4
  2. Extracted specific pieces of information using regular expressions
  3. Stored the data in a Python Pandas dataframe
  4. Converted the dataframe into a comma seperated file

Task 2: parking appeal similarity

The next task was my main task, which was to develop a machine learning model to identify semantically similar carpark appeals. This model will be used to assist URA carpark officers in evaluating parking offense appeal requests, reducing their workload. The steps that I used to calculate text similarity were as follows:

  1. From an archive of parking offense appeal e-mails, obtain two e-mails in the form of text files
  2. using the pre trained models, convert the text into sentence embeddings
  3. calculate cosine similarity score for the two text embeddings obtained
The cosine similarity scores obtained can tell us if the two parking offense appeals are semantically similar or not.
